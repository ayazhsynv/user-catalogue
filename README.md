# Kiçik İstifadəçi Kataloqu

## Istifadə Olunanlar

- React JSX
- CSS
- API kimi JSON-Server ilə db.json (PORT 8000)

## Funksionallıq

- Userların Cədvəl Şəklində çıxarışı
- Userların Ada, Emailə, Rola görə axtarışı və sıralanması
- Userin əlavə olunması, məlumatlarının dəyişdirilməsi və silinməsi
- Formlarda Validation
